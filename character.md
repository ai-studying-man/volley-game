# Retro Volleyball Character Design Brief

## Project Goal
Create six selectable retro-style volleyball game characters inspired by the cast vibe of the Korean variety show *Infinite Challenge*.

The game should feel like a nostalgic 1990s arcade sports game, similar in spirit to classic two-player volleyball games, but with original pixel-art characters designed for mobile gameplay.

## Important Legal / Style Guidance
- Do **not** copy, trace, or directly reproduce broadcast screenshots.
- Do **not** use official *Infinite Challenge* logos, captions, costumes, or copyrighted stills as final assets.
- Use uploaded reference images only as inspiration for silhouette, expression, energy, and comedic personality.
- Final characters should be **retro caricature-inspired**, not photorealistic portraits.
- Avoid using real names in the final commercial build if licensing is not secured. Use nickname-style character names instead.

---

# Global Art Direction

## Visual Style
- 16-bit / 32-bit Korean arcade game aesthetic
- Chunky pixel-art characters
- Large expressive heads, small athletic bodies
- Bold outline, readable silhouette
- Slightly exaggerated facial expressions
- Bright but limited color palette
- Comedic sports-anime energy
- Mobile-friendly readability at small size

## Canvas / Sprite Requirements
- Character selection portrait: 512x512 px
- In-game sprite sheet: 128x128 px per frame
- Idle animation: 4 frames
- Run animation: 6 frames
- Jump animation: 3 frames
- Spike animation: 4 frames
- Hit / receive animation: 3 frames
- Victory pose: 2 frames
- Defeat pose: 2 frames

## Camera / Gameplay Readability
- Characters must be distinguishable from a distance.
- Head shape, hairstyle, outfit color, and posture should be unique.
- Avoid tiny accessories that disappear on mobile.
- Use clear silhouettes and high-contrast colors.

## Shared Outfit Theme
All six characters should wear fictional retro volleyball uniforms.

Suggested shared elements:
- 1990s sports headbands
- Short-sleeve jersey
- Knee pads
- Sneakers
- Numbered jerseys
- Slightly comedic proportions
